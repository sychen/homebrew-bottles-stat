homebrew-bottles-stat
=====================

Calculate the number of bottles in homebrew along time

# Desciption

This program makes use of the history of Homebrew git repository
to calculate the number of bottles available in a certain release on a certain day.

# Usage

  python get-homebrew-bottle-data.py

A CSV with fields Date, Total, and every OSX release from latest are generated.

# Result

A result on 2014/11/22 is available at http://sychen.logdown.com/posts/2014/11/22/how-fast-do-yosemite-bottles-appear-in-homebrew
